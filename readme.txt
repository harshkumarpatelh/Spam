The "Spam.py" is source code for deployment and is ready to predict for spam or not spam.
The "Spam using Naive Bayes.ipynb" is source code for implementation of naive bayes algorithm to build the model.
The "spam.csv" is dataset to build the model.
The "C_vector.pkl" is object of 'CountVectorizer' of model that has already trained on dataset.
The "Spam_model.pkl" is trained model loaded.
